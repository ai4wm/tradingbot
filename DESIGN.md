# Trading Bot Design System

## 1. Atmosphere & Identity

영웅문 HTS를 닮은 고밀도 데스크톱 정보 화면이다. 장식보다 빠른 판독을 우선하며, 종목 상태를 글자색과 셀 모서리 표식으로 일관되게 전달한다.

## 2. Color

| 역할 | 값 | 용도 |
|---|---|---|
| 상승 | `#e83030` | 상승 가격·등락률 |
| 하락 | `#2050d0` | 하락 가격·등락률 |
| 코스닥 | `#C080F0` | 코스닥 종목명 |
| 관리종목 | `#FF6A3D` | 관리종목명, 코스닥보다 우선 |
| NXT | `#FFDD00` | 종목명 셀 좌상단 표식 |
| 미수가능 | `#33C24D` | 종목명 셀 우상단 표식 |
| 신규 당일 | `#FF3DC8` | 종목명 셀 좌하단 표식 |
| 신규 15일 | `#38B8FF` | 종목명 셀 좌하단 표식 |
| 신규 30일 | `#8098B8` | 종목명 셀 좌하단 표식 |

색상 정의는 `gui.py`가 단일 소스이며 다른 화면은 이를 가져다 쓴다.

## 3. Typography

- 전역 글꼴: 굴림체 9pt, 안티앨리어싱 없음.
- 표 헤더와 데이터는 좁은 열에서도 앞부분을 읽을 수 있게 왼쪽 또는 숫자 오른쪽 정렬을 유지한다.

## 4. Spacing & Layout

- 기본 간격 단위는 4px이다.
- 표 행 높이는 22px, 종목명 모서리 표식은 10px 삼각형이다.
- 열 너비와 창 크기는 사용자가 조절하며 `layout.ini`에 보존한다.

## 5. Components

### 종목명 셀 (`NameDelegate`)

- 기본 상태: 거래소 종목명은 기본 글자색, 코스닥은 보라, 관리종목은 주황빨강이다.
- 좌상단: 노랑은 NXT 거래가능이다.
- 우상단: 녹색은 미수가능이다.
- 좌하단: 마젠타·하늘·청회는 각각 상장 당일·15일 이내·16~30일이다.
- 상태가 겹치면 해당 모서리 표식을 함께 표시한다.
- 조건검색과 조회순위의 종목명 열은 같은 델리게이트와 역할 값을 사용한다.
- 툴팁은 모서리 표식 의미를 텍스트로 제공한다.

### 데이터 표

- 행 선택, 읽기 전용, 사용자 열 크기 조절을 지원한다.
- 등락 부호와 상·하한 배경색은 모든 표에서 동일하다.
- 빈 데이터는 빈 표로 표시하고 별도 장식을 추가하지 않는다.

## 6. Motion & Interaction

- 데이터 갱신은 모델 변경 신호로 즉시 반영하며 장식 애니메이션은 사용하지 않는다.
- 종목명 클릭은 종목코드 복사, 우클릭은 종목 정보 문맥 동작을 유지한다.

## 7. Depth & Surface

Qt 기본 표 경계와 선택 강조를 사용한다. 그림자나 별도 카드 표면은 추가하지 않는다.

## 8. Accessibility Constraints & Accepted Debt

- 색만으로 구분되는 모서리 표식에는 동일 의미의 툴팁을 제공한다.
- 선택 상태는 Qt 팔레트의 기본 강조색을 보존한다.
- accepted debt: 키보드만으로 툴팁 내용을 확인하는 별도 경로는 현재 없다. 기존 HTS형 데스크톱 표 동작을 유지하며, 키보드 접근성 요구가 생기면 셀 상세 패널을 추가한다.
